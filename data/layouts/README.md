# Датасет схем фабрик Endfield

## Цель

Собрать примеры оптимальных схем от игроков чтобы Claude научился:
- Понимать паттерны размещения станков
- Оптимально прокладывать конвейеры
- Генерировать схемы под конкретные рецепты

## Как помочь

1. Открой редактор: https://mokhnatti.github.io/endfield-factory-planner/layout_v2.html
2. Создай свою лучшую схему
3. Нажми "Экспорт"
4. Отправь JSON файл Рамилю

## Что нужно в схемах

- Рабочие схемы (не теория, а то что работает в игре)
- Оптимизированные по площади или пропускной способности
- Разные рецепты (бутылки, консервы, капсулы и т.д.)
- Комментарий что схема делает (в названии)

## Формат файла

Редактор сам создаёт правильный JSON. Пример:

```json
{
  "name": "Бутылки 4→2",
  "author": "Nick",
  "facilities": [...],
  "conveyors": [...]
}
```
